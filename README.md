# 📰 Blog Website

A modern blog web application built with Laravel.  
Users can register, login, read blogs, and admins can manage content.

---

## 🚀 Features

- User Registration & Login
- Password Reset (Email)
- Role-based Access (Admin / User)
- Blog Post CRUD (Create, Read, Update, Delete)
- Category Management
- Admin Panel (Custom Dashboard)
- Notification System (New user registration alerts)
- Responsive UI with Tailwind CSS

---

## 🛠️ Tech Stack

- Backend: Laravel 12
- Frontend: Blade + Livewire
- Styling: Tailwind CSS
- Database: MySQL

## 📦 Installation

1. Clone the repository

```bash

2. Install dependencies
composer install
npm install

3. Setup environment
cp .env.example .env
php artisan key:generate

4. Configure database in .env

DB_DATABASE=blog_db
DB_USERNAME=root
DB_PASSWORD=

5. Run migrations
composer run dev
